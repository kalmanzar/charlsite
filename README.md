# charlsite

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

## Netlify Deploy

```bash
# install dependencies (if not done already)
$ npm install

# generate dist dir
$ npm run generate

# deploy draft site
$ netlify deploy

# deploy prod site
$ netlify deploy --prod
```
